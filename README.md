# WoR - World of Robots Assignment
Repo containing my assignment for WoR-World and WoR-Robot.

##Build requirements:
- ROS-Lunar
- OpenCV 3
- catkin
- c++ 11

##Build instructions
- Clone repo to a catkin workspace.

- Build all packages:
 bash: catkin_make

- Build specific package:
 bash catkin_make --pkg <package-name>
